The Event Management System is a web-based application developed using Python and Django that allows users to browse, register, and book events online.
It provides an admin panel to manage events, users, and bookings efficiently.
This project demonstrates full-stack web development using Django framework with authentication and database integration.

## Features
## User Registration & Login Authentication

 ##User Profile Management
  Create, Update, Delete Events (Admin Access)
  Search and Filter Events
  Event Booking System
  View Booking History
  Django Admin Dashboard
  Responsive UI (HTML, CSS, Bootstrap)

 ##Tech Stack
  Backend: Python, Django
  Frontend: HTML, CSS, Bootstrap
  Database: SQLite
  Authentication: Django Built-in Auth System
